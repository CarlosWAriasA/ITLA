Crear un proceso de inscripcion:
1- Debe tener un formulario que valide todos los datos.
  Los campos que debe tener son: nombre, provincia, ciudad, sector, calle y carrera.
  Este ultimo debe permitir seleccionar 3 carreras diferentes.
  Debe tener un boton que permita limpiar los campos y un boton que permita continuar
  al siguiente proceso.
2- Una vez validados todos los datos cuando se le haga click al boton de continuar
  deben aparecer 5 materias segun la carrera elegida con 3 horarios diferentes, no es 
  necesario que se valide si se selecciono un horario o no.
  Debe tener un boton para volver atras al formulario y que le aparezca con todos los
  datos que tenia y un boton de continuar para ir al siguiente proceso
3- Debe mostrar como un menu de confirmacion apareciendo los campos del formulario
  con sus datos y una tabla que se genere a partir de los horarios seleccionados.
  Debe tener un boton que permita volver atras al anterior proceso.
  Debe tener un boton para confirmar si terminar el proceso de inscripcion si o no.
